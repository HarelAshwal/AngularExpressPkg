# Angular + Express + Pkg + Bootstrap + WebSockets

## Installation

1. install pkg globally:
   ``
   npm install -g pkg
   ``

2. clone the repo
   ``
   git clone https://github.com/HarelAshwal/AngularExpressPkg.git
   ``

3. install dependencies:
   ``
   cd AngularExpressPkg
   npm i
   cd ./server & npm i
   ``
4. build package:
   ``
   pkg .
   ``

you're done!
